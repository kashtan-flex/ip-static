1. Общие положения

Настоящая Политика определяет порядок обработки и защиты персональных данных пользователей сайта иванперцев.рф.

Оператор обрабатывает персональные данные в соответствии с Федеральным законом от 27.07.2006 №152-ФЗ «О персональных данных».

2. Какие персональные данные обрабатываются

Через форму обратной связи на сайте Оператор обрабатывает:

  • имя;
  • номер телефона;
  • дату мероприятия.

Также пользователь может самостоятельно сообщить дополнительные сведения при дальнейшем общении с Оператором.

Сайт не собирает персональные данные для рекламных рассылок, ретаргетинга, маркетинговой аналитики и поведенческой рекламы.

Сайт не использует сервисы веб-аналитики и рекламные пиксели, если они фактически не подключены.

2.1. Использование файлов cookie

Сайт использует файлы cookie для корректной работы страниц, воспроизведения видео и сохранения пользовательских настроек интерфейса.

Cookie не используются для рекламных рассылок, ретаргетинга и поведенческой рекламы, если такие сервисы фактически
не подключены.

Пользователь может ограничить или
удалить cookie в настройках браузера.
Это может повлиять на корректность
работы отдельных функций сайта.

3. Цели обработки персональных данных

Персональные данные обрабатываются только для следующих целей:

  • получение заявки от пользователя;
  • связь с пользователем по указанному номеру телефона;
  • уточнение даты, формата и деталей мероприятия;
  • подготовка предложения по услугам ведущего мероприятий;
  • согласование условий сотрудничества;
  • заключение и исполнение договора, если пользователь принимает предложение Оператора.

4. Правовое основание обработки

Правовым основанием обработки персональных данных служит согласие пользователя на обработку персональных данных.

Если после обращения пользователь заключает договор с Оператором, обработка также может выполняться для заключения и исполнения договора.

5. Как обрабатываются персональные данные

Оператор может совершать с персональными данными следующие действия:

  • сбор;
  • запись;
  • хранение;
  • уточнение;
  • использование;
  • передача в рамках технической доставки заявки;
  • блокирование;
  • удаление;
  • уничтожение.

Обработка может выполняться с использованием средств автоматизации и без их использования.

6. Передача данных через электронную почту

После отправки формы обратной связи данные пользователя могут поступить владельцу сайта на электронную почту.

Передаются только данные, которые пользователь указал в форме: имя, номер телефона и дата мероприятия.

Электронная почта используется как канал доставки заявки владельцу сайта. Данные не передаются через электронную почту для рекламных рассылок, массовых сообщений, ретаргетинга или продажи третьим лицам.

7. Передача данных третьим лицам

Оператор не продает персональные данные пользователей и не передает их третьим лицам для самостоятельного использования.

Передача персональных данных возможна только:

  • сервисам, которые технически обеспечивают работу сайта, формы обратной связи и доставку заявки;
  • лицам, которые помогают Оператору отвечать на заявку или исполнять договор;
  • государственным органам, если такая обязанность предусмотрена законом.

8. Срок хранения персональных данных

Персональные данные хранятся до достижения цели обработки.

Если заявка не привела к заключению договора, данные могут храниться в течение срока, необходимого для обработки обращения и деловой переписки.

Если с пользователем заключен договор, данные могут храниться в течение сроков, предусмотренных законодательством РФ для договорных, бухгалтерских и налоговых документов.

9. Отзыв согласия

Пользователь может отозвать согласие на обработку персональных данных.

Для отзыва согласия пользователь может направить обращение Оператору по контактам, указанным на сайте.

После получения отзыва Оператор прекращает обработку персональных данных, кроме случаев, когда дальнейшая обработка допускается законом.

10. Права пользователя

Пользователь вправе:

  • получать информацию об обработке своих персональных данных;
  • требовать уточнения персональных данных;
  • требовать блокирования или удаления персональных данных;
  • отзывать согласие на обработку персональных данных;
  • направлять Оператору запросы по вопросам обработки персональных данных.

11. Защита персональных данных

Оператор принимает необходимые меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия, уничтожения и иных неправомерных действий.

Доступ к персональным данным имеют только Оператор и лица, которым такие данные нужны для обработки заявки или исполнения договора.

12. Изменение Политики

Оператор может изменять настоящую Политику.

Актуальная редакция Политики размещается на сайте иванперцев.рф.
